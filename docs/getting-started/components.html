Components

GenesisUI based on Bootstrap 4, but we use also other frameworks, plugins and components.

Main Components

  1. Bootstrap 4 - http://v4-alpha.getbootstrap.com
  2. jQuery - https://jquery.com
  3. Tether - http://github.hubspot.com/tether/
  4. AngularJS - https://angularjs.org
  5. AngularUI Router - http://angular-ui.github.io/ui-router/site/#/api/ui.router
  6. ocLazyLoad - https://oclazyload.readme.io
  7. Angular Breadcrumb - http://ncuillery.github.io/angular-breadcrumb/#/home
